#                                    shadowsocks
使用方法：

    wget --no-check-certificate https://github.com/52fancy/shadowsocks/raw/master/ss.sh && bash ss.sh
或者

    wget --no-check-certificate https://github.com/52fancy/shadowsocks/raw/master/ssr.sh && bash ssr.sh
